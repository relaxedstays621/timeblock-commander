## First Steps

1. Read `../INDEX.md` to understand the project documentation map.
2. Read `../HOW_TO_USE.md` to choose the right beginner path.
3. Fill in the local setup section below.
4. If an LLM will work on the project, send it to `../Purpose/README.md`.
